# **App Name**: City School Hub

## Core Features:

- User Authentication: Secure authentication using Google login and email/password options.
- Role-Based Access: Differentiate access and features based on user roles (Teacher and Student).
- Real-Time Video Classrooms: Implement live video sessions using the Daily.co API (or Daily library).
- Live Q&A: Enable students to post questions during live sessions, visible to all participants.
- Temporary Chat System: Allow students and teachers to exchange messages that auto-delete after 30 minutes.
- Class Creation and Management: Teachers can create and manage online classes.
- File Upload: Teachers can upload files (max 20MB) for students to download.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke trust and learning.
- Background color: Light blue (#E6F3FF), almost white but slightly tinted for depth.
- Accent color: Soft purple (#D8BFD8) for interactive elements.
- Body font: 'PT Sans', a humanist sans-serif with a modern, approachable feel, for readability in both headlines and body text.
- Use blueprint-style icons, simple and clear, related to education and classroom activities.
- Responsive and clean layout with a focus on accessibility.
- Subtle animations for user feedback, like button presses and loading states.
